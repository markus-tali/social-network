## Project Description

Networking is a social network project that aims to connect people in a unique way. Built with a modern tech stack, it offers a seamless and engaging user experience.

### Tech Stack:

- **Frontend**: React
- **Backend**: Golang
- **Database**: SQLite

## How to Run the Project

### Run Locally

To run the project locally, follow these steps:

#### Prerequisites

- Golang installed
- Node.js and npm installed

#### Steps

1. **Clone the repository**:

   ```bash
   git clone https://01.kood.tech/git/mtali/social-network
   cd networking
   ```

2. **Run the backend**:

   - Open the first terminal

   ```bash
   cd backend
   go get
   go run .
   ```

3. **Run the frontend**:
   - Open the second terminal
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
